# transformers Modified Version
This repository contains modified files based on [transformers](https://github.com/huggingface/transformers) (Apache 2.0 License), adding new feature that  loading  extracted feature directly.

## Key Modifications
| File Path | Modification Description |
|-----------|--------------------------|
| transformers/models/qwen2_5_omni/processing_qwen2_5_omni.py | Added support for  loading extracted feature directly|

## How to Use
1. Clone the original LlamaFactory repository:
   ```bash
   git clone https://github.com/huggingface/transformers
   cd transformers
   ```
2. Replace the corresponding files in the original repository with the modified files in this repo.
